# 自传式记忆巩固设计

## 1. 目标

自传式记忆巩固的目标是把单次任务后的临时变化，逐步压缩成长期可解释的身份叙事。

## 2. 当前机制

当前版本已经支持：

1. 每次运行时更新生成一条 autobiographical event
2. 将事件摘要写入近期身份更新列表
3. 增量更新 long-term narrative
4. 对高矛盾事件提高记忆显著性

## 3. 当前价值

这一步让系统开始拥有“我经历过什么”的连续记录，而不是只保存当前状态快照。

## 4. 当前限制

- 还是启发式压缩
- 还没有主题聚类
- 还没有睡眠期离线重写
- 还没有关系记忆整合

## 5. 下一步

- 引入离线 consolidation job
- 将多条事件压缩为阶段性生命史摘要
- 把社会关系变化合并进 autobiographical memory
