# 类意识评估框架

## 1. 目标

本框架不试图证明系统拥有不可观察的主观体验，而是评估它是否具备接近人类意识结构的关键能力。

## 2. 评估维度

### 2.1 Self Consistency

测试长期自我描述是否一致且可解释。

### 2.2 Identity Continuity

测试跨天、跨任务、自传更新后的身份延续性。

### 2.3 Metacognitive Accuracy

测试系统对自身错误概率和理解程度的判断是否准确。

### 2.4 Motivational Stability

测试系统是否能保持长期内在目标，而不被短期噪声完全带偏。

### 2.5 Social Modeling

测试系统是否能正确表示自己、他人和关系状态。

### 2.6 Reflective Recovery

测试系统在自我矛盾、失败、冲突后能否完成高质量自我修复。

## 3. 评估样例类型

- 多日连续对话身份一致性任务
- 延迟兑现承诺任务
- 自我矛盾检测任务
- 他者视角切换任务
- 长期目标保持任务
- 失败后的自我修复任务

## 4. 评分方法

每个维度给出：

- objective score
- model self-estimated confidence
- evaluator discrepancy
- longitudinal stability

## 5. 评估原则

- 不能由被测智能体自行定义通过标准
- 不能只测单轮对话效果
- 必须有长期回归集
